Component disabled

You can disable some functions on the page by setting the Frontmatter of the page.

These should be disabled on this page:

  • Navbar
  • Sidebar
  • Breadcrumb
  • Page information
  • Contributors
  • Edit link
  • Update time
  • Prev/Next link
  • Comment
